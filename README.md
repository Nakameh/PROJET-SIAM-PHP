## Installation

Pour installer le projet, il suffit de cloner le dépôt git et de lancer un serveur PHP.

```bash
git clone
cd siam
php -S localhost:8000
```

## Utilisation

Pour utiliser le projet, il suffit de se rendre sur la page d'accueil du projet.

```bash
http://localhost:8000
```
